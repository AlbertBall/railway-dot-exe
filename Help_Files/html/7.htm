7  Utilities and Further Development

 

Users have developed several utilities to run in conjunction with RailOS.  These are available on the official RailOS website at https://www.railwayoperationsimulator.com/pc/utilities. 

 

In addition Krizar has built a GitHub site for use by those wishing to develop railway projects, utilities and tools in a controlled environment.  The URL is https://github.com/Railway-Op-Sim. 

 

Also for those who prefer a traditional program installer Krizar has developed one that runs under Chocolatey.  It is available at https://community.chocolatey.org/packages/railwayopsim

 

The program itself is open source and available for further development at https://github.com/AlbertBall/railway-dot-exe
